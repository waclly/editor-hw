## How to build and run

1. Clone the repository:

   ```bash
   git clone https://github.com/waclly/editor-hw.git

2. Change into the project directory:
   ```bash
   cd editor-hw

3. Build the project:
   ```bash
   make

4. Run the checker or validator:
   ```bash
    ./checker
    # or
    ./validator "<Design><Code>hello world</Code></Design>"

5. easy deploy:
    ```bash
    git clone https://github.com/waclly/editor-hw.git
    cd editor-hw
    make
    ./checker
